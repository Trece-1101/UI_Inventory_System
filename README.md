<!-- 2. Description -->
Un sistema de inventario completo y modular diseñado en Unreal al mejor estilo Skyrim. El sistema permite gestionar objetos, equipamiento y almacenamiento con una interfaz de usuario dinámica y "Drag & Drop".

**Principales características:**
* **Drag & Drop:** Mover objetos entre celdas e inventarios.
* **Stacking:** Apilamiento de objetos consumibles.
* **Equipment Slots:** Ranuras específicas para armas, armaduras, etc.
* **Tooltips:** Información detallada al pasar el mouse.

---

***Engine:*** Unreal v5.4

***Lenguaje:*** Blueprints

***Plataforma:*** PC
